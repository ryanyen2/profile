# Ryanyen2 Profolio [![GitHub](https://img.shields.io/github/license/cobidev/gatsby-simplefolio?color=blue)](https://github.com/cobidev/gatsby-simplefolio/blob/master/LICENSE.md))

## Features

- Modern UI Design + Reveal Animations\
- One Page Layout built with React\
- Styled with Bootstrap v4.3 + Custom SCSS\
- Fully Responsive\
- Configurable color scheme\
- Image optimization with Gatsby\
- Easy site customization\
- Well organized documentation

## Build Up Tools

- [Gatsby](https://www.gatsbyjs.org/) - Static Site Generator
- [GraphQL](https://graphql.org/) - Query language for APIs
- [React](https://es.reactjs.org/) - Front-End JavaScript library
- [Bootstrap 4](https://getbootstrap.com/docs/4.3/getting-started/introduction/) - Front-End UI library
- [Sass](https://sass-lang.com/documentation) - CSS extension language
